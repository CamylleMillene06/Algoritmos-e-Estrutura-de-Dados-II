# Código-fonte

Esta pasta contém os arquivos responsáveis pela implementação do projeto de análise da rede de pré-requisitos do NICEN.

Arquivos:

- `main.py` – arquivo principal responsável pela execução do programa.
- `grafo.py` – implementação da estrutura do grafo e funções auxiliares.
- `bfs.py` – implementação do algoritmo de Busca em Largura (BFS).
- `dfs.py` – implementação do algoritmo de Busca em Profundidade (DFS).
- `componentes.py` – implementação do algoritmo de Kosaraju para identificação das componentes fortemente conexas.
